## 本地运行

1. 安装依赖：
   - `npm install`
2. 启动开发服务器（会自动同步桌面样本数据）：
   - `npm run dev`

## 样本数据说明

- 页面“结果页”内容来自你桌面目录 `~/Desktop/essay sample/`
- 启动/构建时会自动同步到 `public/essay-sample/`，前端通过 `fetch` 读取展示
